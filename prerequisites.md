# Installing

## Using brew

```bash
brew install jq curl helm helmfile  az
helm plugin install https://github.com/databus23/helm-diff
```

## Using chocolatey

```powershell
choco install azure-cli jq kubernetes-cli kubernetes-helm  sonobuoy
choco install fleet
choco install azure-kubelogin
choco install k9s
choco install arkade
choco install ekz
choco install docker-cli
choco install docker-desktop
choco install tanzu-community-edition
choco install curl
choco install gotop
choco install shfmt
helm plugin install https://github.com/databus23/helm-diff
```
